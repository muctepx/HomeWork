Написать программу, которая из имеющегося массива строк формирует массив строк, длина которых меньше либо равна 3 символа. Первоначальный массив можно ввсти с клавиатуры, либо задать на старте выполнения алгоритма.

Алгоритм программы:
Предлагаем пользователю ввести массив данных через запяту без пробелов.
Создаем переменную, которую получаем путем считывания данных, введеннх пользователем.
Создаем массив, элементы которого, формируются из созданной переменной, а разделителем для определения элеметов, является запятая без пробела.
Создаем итоговый массив, в который попадают данные, с длинной каждого элемента менее 4 символов.
Распечатываем полученные элементы массива через запятую.
